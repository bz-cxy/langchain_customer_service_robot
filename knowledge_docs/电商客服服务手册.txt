================================================================================
                          电商客服服务手册
================================================================================

一、退货政策
--------------------------------------------------------------------------------
1. 退货时效
   - 自商品签收之日起7天内可申请无理由退货
   - 15天内可因质量问题退货
   - 超过15天但仍在保修期内的商品，可申请维修服务

2. 退货条件
   - 无理由退货：商品完好、配件齐全、未拆封/未使用（贴身用品、定制商品除外）
   - 质量问题退货：需提供质量检测报告或清晰的问题照片/视频
   - 商品原包装完好，不影响二次销售

3. 不支持退货的商品
   - 食品、化妆品等一次性消耗品
   - 定制类商品（如刻字礼品、定制家具）
   - 贴身用品（如内衣、袜子）
   - 已拆封的软件、音像制品
   - 清仓特卖商品
   - 赠品

4. 退货注意事项
   - 请保留商品原包装和所有配件
   - 贵重物品建议使用保价快递
   - 退货前请先联系客服获取退货地址


二、退货流程
--------------------------------------------------------------------------------
1. 申请退货
   - 登录会员中心
   - 进入"我的订单"
   - 选择对应订单
   - 点击"申请退货"
   - 填写退货原因并上传凭证

2. 审核阶段
   - 客服将在1-2个工作日内审核申请
   - 审核通过后发送退货地址
   - 审核不通过会说明原因

3. 寄回商品
   - 48小时内将商品寄回指定地址
   - 运费说明：
     * 质量问题由商家承担
     * 无理由退货由用户承担

4. 验收退款
   - 仓库验收商品无误后，1-3个工作日内完成退款
   - 退款路径：按原支付方式原路退回
     * 微信/支付宝：即时到账
     * 银行卡：1-3个工作日到账


三、换货政策
--------------------------------------------------------------------------------
1. 换货时效
   - 自商品签收之日起15天内可申请换货
   - 换货商品需在库存范围内

2. 换货条件
   - 商品存在质量问题（如破损、功能故障）
   - 非人为损坏，且配件、包装齐全
   - 不支持无理由换货，仅针对质量问题/错发漏发

3. 换货范围
   - 仅支持更换同款式、同规格的商品
   - 不支持更换其他款式/规格
   - 如需更换其他款式，建议申请退货后重新购买

4. 特殊情况处理
   - 换货商品缺货：可选择退款或等待补货
   - 换货商品价格变动：按原订单价格处理


四、换货流程
--------------------------------------------------------------------------------
1. 申请换货
   - 联系在线客服
   - 提供订单号+问题描述+凭证（照片/视频）
   - 客服登记换货需求

2. 审核阶段
   - 1个工作日内审核
   - 审核通过后告知换货方式

3. 寄回/补发
   - 方式1：先寄回问题商品，仓库验收后补发新商品（运费商家承担）
   - 方式2：加急换货可先补发新商品，用户收到后48小时内寄回问题商品

4. 完成确认
   - 新商品寄出后会发送物流单号
   - 确认收货后换货流程完成


五、退款政策
--------------------------------------------------------------------------------
1. 退款时效
   - 验收商品无误后，1-3个工作日内完成退款
   - 特殊情况可能延长至5个工作日

2. 退款金额
   - 无理由退货：全额退款（不含运费）
   - 质量问题退货：全额退款+承担寄回运费
   - 部分退货：按实际退货商品金额退款

3. 特殊情况
   - 促销商品退货：需扣除已使用的优惠券金额
   - 分期付款商品：退款后分期账单自动取消
   - 使用积分抵扣：积分将原路退回

4. 退款路径
   - 微信支付：原路退回，即时到账
   - 支付宝：原路退回，即时到账
   - 银行卡：1-3个工作日到账
   - 余额支付：退回账户余额


六、物流服务
--------------------------------------------------------------------------------
1. 配送时效
   - 一线城市：1-2个工作日
   - 二三线城市：2-4个工作日
   - 偏远地区：5-7个工作日
   - 大促期间可能延迟1-2天

2. 物流查询
   - 登录会员中心查看订单状态
   - 点击"查看物流"获取实时信息
   - 可拨打快递公司客服电话查询

3. 签收注意事项
   - 请当面验货后再签收
   - 如发现包装破损，可拒收并联系客服
   - 贵重商品建议本人签收

4. 物流问题处理
   - 包裹丢失：联系客服申请补发或退款
   - 包裹破损：提供照片凭证，申请售后
   - 配送延迟：联系物流公司或客服查询


七、支付方式
--------------------------------------------------------------------------------
1. 支持的支付方式
   - 微信支付
   - 支付宝
   - 银行卡（借记卡/信用卡）
   - 账户余额
   - 分期付款（花呗、信用卡分期）

2. 支付安全
   - 所有支付均通过安全加密通道
   - 不会存储您的银行卡密码
   - 如遇支付异常，请联系客服

3. 发票服务
   - 可在订单确认页面申请电子发票
   - 增值税专用发票需提供企业资质
   - 发票开出后不可修改


八、会员服务
--------------------------------------------------------------------------------
1. 会员等级
   - 普通会员：注册即可成为普通会员
   - 银卡会员：累计消费满500元
   - 金卡会员：累计消费满2000元
   - 钻石会员：累计消费满5000元

2. 会员权益
   - 普通会员：享受基础服务
   - 银卡会员：生日礼包、专属客服
   - 金卡会员：9.5折优惠、优先发货
   - 钻石会员：9折优惠、免费退换货、专属客服经理

3. 积分规则
   - 消费1元=1积分
   - 积分可用于抵扣现金（100积分=1元）
   - 积分有效期为2年


九、常见问题解答
--------------------------------------------------------------------------------
1. 退货后运费何时返还？
   答：质量问题退货的运费，在退款完成后24小时内返还至会员余额。

2. 换货可以更换不同规格吗？
   答：不可以，仅支持更换同款式、同规格的商品。

3. 超过7天还能无理由退货吗？
   答：不可以，无理由退货仅限签收后7天内，超过7天仅支持质量问题退货。

4. 退款迟迟不到账怎么办？
   答：可联系在线客服提供订单号查询，或拨打售后热线400-888-XXXX。

5. 订单已发货可以修改地址吗？
   答：订单发货后无法修改地址，建议联系快递公司转寄或联系客服协助处理。

6. 商品少件/错发怎么办？
   答：请在签收后24小时内联系客服，提供订单号和照片凭证，我们将尽快处理。

7. 如何申请发票？
   答：可在订单详情页申请电子发票，或联系客服开具纸质发票。

8. 优惠券可以退回吗？
   答：使用优惠券的订单如发生退货，优惠券不予退回（特殊情况除外）。

9. 支持货到付款吗？
   答：部分地区支持货到付款，具体以下单页面显示为准。

10. 如何投诉或建议？
    答：可拨打客服热线400-888-XXXX，或通过会员中心提交意见反馈。
